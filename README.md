# Crash-Prediction-Model
Machine learning project for road crash severity prediction using Logistic Regression and Decision Tree models, with data preprocessing, exploratory data analysis, and predictive modeling on driver crash reporting data
Crash Severity Prediction using Machine Learning

This project applies machine learning techniques to analyze road crash data and predict accident severity based on driver behavior, environmental conditions, and traffic-related factors.

Project Objectives
1. Injury Severity Prediction
Predict whether a crash will result in an injury.
Implemented using Logistic Regression.
Evaluated using accuracy, confusion matrix, and classification metrics.
2. Severe/Fatal Crash Prediction
Classify crashes as severe/fatal or non-severe.
Implemented using a Decision Tree Classifier.
Identifies key factors contributing to high-risk accidents.
Features
